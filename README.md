# Book Vibe by Tahmid Tausif


[Book-Vibe](https://b9a8-book-vibe-tahmidtausif.netlify.app/)

This is a book store website which lets you list your favourite books in either read list or wish list. You can see a books details and reviews about the book. There is also a chart which showcases your read list books number of pages. To build this website I've used -

-[React](#React)

-[Tailwindcss](#Tailwindcss)

-[Daisyui](#Daisyui)

-[React-router](#React-router)

-[Javascript](#Javascript)
